# Generate the FCM token

## 1. Add the server key to the .env file

```FCM_SERVER_KEY=XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX```

## 2. Install the dependencies

```bash 
composer install
npm install
vite build
```

## 3. Run the appplication to generate the FCM token

```bash
php artisan serve
```
